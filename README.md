# Bingo Scheduler

Simple, mobile-friendly bingo event scheduler for GitHub Pages frontend + Google Apps Script backend + Google Sheets datastore.

## Pages
- `index.html` - user sign-in/profile entry
- `events.html` - user events and my sign-ups
- `admin.html` - admin login + dashboard

## Backend
- `apps-script/Code.gs` - Apps Script action router and business logic

See `docs/setup.md` for full deployment/setup steps.
